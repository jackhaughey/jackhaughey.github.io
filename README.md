# Cloud Engineering Notes — Kubernetes & Azure

A minimal Jekyll-powered blog used as a personal knowledge base for continuous learning in Kubernetes, Azure, and cloud engineering.  
The site is intentionally simple: Markdown notes, a single layout, and GitHub Pages for hosting.

## Purpose

This repository serves as a long-term learning log covering:

- Kubernetes fundamentals, patterns, and troubleshooting
- Azure architecture, services, and CLI workflows
- Cloud-native tooling (kubectl, Helm, Terraform, Azure CLI)
- Practical notes, commands, diagrams, and experiments

All notes are written in Markdown and published automatically via GitHub Pages.

## Structure
```
├── _config.yml
├── _layouts/
│   └── default.html
├── _posts/
│   └── YYYY-MM-DD-topic.md
└── index.md
```

Each note lives in `_posts/` and is timestamped for chronological learning progression.

## Creating a New Note

Add a Markdown file to `_posts/`:
```YYYY-MM-DD-topic.md```
